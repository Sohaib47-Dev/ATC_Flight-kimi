------------------------------------------------------------------
1) PROJECT IDEA IN SIMPLE WORDS
------------------------------------------------------------------

This is a Flask web app for ATC + Defense workflow:

1. ATC stores or retrieves a flight plan by CALLSIGN.
2. System parses ICAO plan text and extracts:
   - callsign
   - aircraft type
   - departure
   - destination
   - fir entry, speed, route
3. ATC enters estimates (ETO/CFL/SSR) to create an active track.
4. Track is encrypted and sent to defense side.
5. Defense sees tracks in dashboard + radar (HTTP and WebSocket updates).
6. Admin can manage users, logs, flight plans, tracks, and data integrity checks.
