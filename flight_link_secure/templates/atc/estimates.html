Validation Rules
ETO (Estimated Time Over)
  • Format: HHMM (UTC)
  • Hours: 00-23
  • Minutes: 00-59
  • Auto-converts to Pakistan Time (UTC+5)
CFL (Cleared Flight Level)
  • Exactly 3 digits
  • Maximum: 500
  • Example: 350 = FL350
  • Rejected: 50, 0500, FL600
SSR Code
  • Exactly 4 digits
  • No letters allowed
  • No repeated digits (1111, 2222)
  • Auto-corrected if invalid
